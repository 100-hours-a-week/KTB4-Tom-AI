# 맛집 검색 CLI

판교 근처 맛집 정보를 터미널에서 검색하고 관리할 수 있는 CLI 프로그램입니다.

## 실행 환경

- Python 3.x
- 추가 패키지 설치 필요 X (표준 라이브러리만 사용)

## 실행 방법

```bash
python3 restaurant.py
```

실행하면 도움말이 출력되고, 명령어를 입력할 수 있는 대화형 모드로 진입합니다.  
종료하려면 `quit`을 입력합니다.

## 파일 구조

```
01/
├── restaurant.py   # 메인 프로그램
├── data.json       # 음식점 데이터 (46곳)
└── README.md
```

## 데이터 구조

각 음식점은 다음과 같은 형식으로 `data.json`에 저장되어 있습니다.

```json
{
  "name": "면학",
  "price": ["10000원~15000원"],
  "likes": 5,
  "category": ["일식", "중식"]
}
```

| 필드 | 설명 |
|------|------|
| name | 음식점 이름 |
| price | 가격대 (리스트) |
| likes | 좋아요 수 |
| category | 카테고리 (리스트) |

## 명령어

### list

모든 음식점의 정보를 출력합니다.

```
list
```

### category \<카테고리\>

해당 카테고리에 속하는 음식점 이름을 출력합니다.

```
category 한식
```

### show_category

등록된 카테고리 종류를 출력합니다.

```
show_category
```

### name \<이름\>

입력한 키워드가 이름에 포함된 음식점 정보를 출력합니다.

```
name 양꼬치
```

### like \<숫자\>

좋아요 수가 입력한 숫자 이상인 음식점을 출력합니다.

```
like 3
```

### popular \<숫자\>

좋아요 수가 많은 순서대로 입력한 개수만큼 출력합니다.

```
popular 5
```

### recommend \<이름\>

해당 음식점의 좋아요 수를 1 올립니다. 이름이 정확히 일치해야 합니다.

```
recommend 면학
```

### new

새로운 음식점을 대화형으로 추가합니다. 가게 이름, 가격대, 카테고리를 순서대로 입력합니다.

```
new
```

### help

사용 가능한 명령어 목록과 설명을 출력합니다.

```
help
```

### quit

프로그램을 종료합니다.

```
quit
```
